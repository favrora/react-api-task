### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/favrora/react-api-task.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Build and run app:
   ```sh
   npm run start
   ```
4. App will run here => [http://localhost:3000](http://localhost:3000)
